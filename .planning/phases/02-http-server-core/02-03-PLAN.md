## WinterTC Specification Reference (from CONTEXT.md)

WinterTC defines Minimum Common Web Platform API for server-side JavaScript:
- https://wintertc.org/working-group/

**Key APIs for this phase:**
- Request: method, url, headers, body
- Response: status, statusText, headers, body
- Headers: append, delete, get, getSetCookie, has, set
- URL: parsing, searchParams, href, origin, etc.
- URLSearchParams: get, getAll, has, set, append, delete, toString

## Decisions from CONTEXT.md

**D-05:** Hybrid body handling — buffer small bodies (<1MB) in memory, stream large bodies
- Implement with enum: Body::InMemory(Bytes) or Body::Stream(Receiver)
- Threshold: 1MB (1048576 bytes)

**D-06:** Response objects via JSON serialization → V8 parse (not direct V8 API creation)
- Serialize Rust response to JSON string
- Parse in V8 with `JSON.parse()`
- This is simpler than building V8 objects directly

**D-07:** Case-insensitive header names, normalized to lowercase per RFC 7230
- Use `HashMap<String, Vec<String>>` with lowercase keys
- All header name operations normalize to lowercase

**D-08:** Multiple values combine with commas, except Set-Cookie which stays separate
- `get()` returns comma-combined values (except Set-Cookie)
- `getSetCookie()` returns array of separate Set-Cookie values

**D-09:** Full WinterTC URL compliance (not basic parsing only)
- Implement all URL properties: href, origin, protocol, host, hostname, port, pathname, search, hash
- URLSearchParams with full API

**D-10:** Percent-decode with lossy UTF-8 replacement (U+FFFD for invalid sequences)
- Use `percent_decode_lossy()` from `percent-encoding` crate
- Invalid UTF-8 sequences become U+FFFD replacement character

   use std::collections::HashMap;
   use percent_encoding::percent_decode_str;
   
   #[derive(Debug, Clone, Default)]
   pub struct NanoUrlSearchParams {
       params: HashMap<String, Vec<String>>,  // name -> values
   }
   
   impl NanoUrlSearchParams {
       pub fn from_query(query: Option<&str>) -> Self {
           let mut params = HashMap::new();
           if let Some(q) = query {
               for pair in q.split('&') {
                   if let Some((key, value)) = pair.split_once('=') {
                       let decoded_key = percent_decode_str(key).decode_utf8_lossy().to_string();
                       let decoded_value = percent_decode_str(value).decode_utf8_lossy().to_string();
                       params.entry(decoded_key).or_insert_with(Vec::new).push(decoded_value);
                   }
               }
           }
           Self { params }
       }
       
       // WinterTC URLSearchParams API
       pub fn get(&self, name: &str) -> Option<String> {
           self.params.get(name).and_then(|v| v.first().cloned())
       }
       
       pub fn get_all(&self, name: &str) -> Vec<String> {
           self.params.get(name).cloned().unwrap_or_default()
       }
       
       pub fn has(&self, name: &str) -> bool {
           self.params.contains_key(name)
       }
       
       pub fn set(&mut self, name: &str, value: &str) {
           self.params.insert(name.to_string(), vec![value.to_string()]);
       }
       
       pub fn append(&mut self, name: &str, value: &str) {
           self.params.entry(name.to_string()).or_default().push(value.to_string());
       }
       
       pub fn delete(&mut self, name: &str) {
           self.params.remove(name);
       }
       
       pub fn to_string(&self) -> String {
           let mut pairs = Vec::new();
           for (key, values) in &self.params {
               for value in values {
                   pairs.push(format!("{}={}", key, value));
               }
           }
           pairs.join("&")
       }
       
       pub fn entries(&self) -> impl Iterator<Item = (&String, &Vec<String>)> {
           self.params.iter()
       }
   }
   ```

   use std::collections::HashMap;
   
   /// WinterTC-compliant Headers implementation
   ///
   /// Case-insensitive header names per RFC 7230 (D-07).
   /// Multiple values combine with commas, except Set-Cookie (D-08).
   #[derive(Debug, Clone, Default)]
   pub struct NanoHeaders {
       headers: HashMap<String, Vec<String>>,  // lowercase name -> values
   }
   
   impl NanoHeaders {
       pub fn new() -> Self {
           Self::default()
       }
       
       /// Create from axum header map
       pub fn from_axum_headers(axum_headers: &axum::http::HeaderMap) -> Self {
           let mut headers = HashMap::new();
           for (name, value) in axum_headers {
               let name_lower = name.as_str().to_lowercase();
               let value_str = value.to_str().unwrap_or("").to_string();
               headers.entry(name_lower).or_insert_with(Vec::new).push(value_str);
           }
           Self { headers }
       }
       
       // WinterTC Headers API
       pub fn get(&self, name: &str) -> Option<String> {
           let name_lower = name.to_lowercase();
           self.headers.get(&name_lower).map(|values| {
               if name_lower == "set-cookie" {
                   // D-08: Set-Cookie stays separate, return first value
                   values.first().cloned().unwrap_or_default()
               } else {
                   // D-08: Other headers combine with commas
                   values.join(", ")
               }
           })
       }
       
       pub fn get_set_cookie(&self) -> Vec<String> {
           // D-08: Returns all Set-Cookie values separately
           self.headers.get("set-cookie").cloned().unwrap_or_default()
       }
       
       pub fn has(&self, name: &str) -> bool {
           self.headers.contains_key(&name.to_lowercase())
       }
       
       pub fn set(&mut self, name: &str, value: &str) {
           // Replace any existing values
           self.headers.insert(name.to_lowercase(), vec![value.to_string()]);
       }
       
       pub fn append(&mut self, name: &str, value: &str) {
           self.headers.entry(name.to_lowercase()).or_default().push(value.to_string());
       }
       
       pub fn delete(&mut self, name: &str) {
           self.headers.remove(&name.to_lowercase());
       }
       
       pub fn for_each<F>(&self, callback: F)
       where
           F: FnMut(&str, &str),
       {
           let mut callback = callback;
           for (name, values) in &self.headers {
               let value = if name == "set-cookie" {
                   values.first().map(String::as_str).unwrap_or("")
               } else {
                   &values.join(", ")
               };
               callback(name, value);
           }
       }
       
       pub fn entries(&self) -> impl Iterator<Item = (&String, &Vec<String>)> {
           self.headers.iter()
       }
       
       pub fn to_axum_headers(&self) -> axum::http::HeaderMap {
           let mut map = axum::http::HeaderMap::new();
           for (name, values) in &self.headers {
               for value in values {
                   if let Ok(name) = axum::http::HeaderName::from_bytes(name.as_bytes()) {
                       if let Ok(value) = axum::http::HeaderValue::from_str(value) {
                           map.append(name, value);
                       }
                   }
               }
           }
           map
       }
   }
   ```

   /// WinterTC-compatible Response object
   ///
   /// Represents an HTTP response returned from the JavaScript handler.
   #[derive(Debug, Clone)]
   pub struct NanoResponse {
       status: u16,
       status_text: String,
       headers: NanoHeaders,
       body: Option<Bytes>,
   }
   
   impl NanoResponse {
       pub fn new(status: u16, headers: NanoHeaders, body: Option<Bytes>) -> Self {
           let status_text = Self::default_status_text(status);
           Self { status, status_text, headers, body }
       }
       
       // WinterTC Response properties
       pub fn status(&self) -> u16 { self.status }
       pub fn status_text(&self) -> &str { &self.status_text }
       pub fn headers(&self) -> &NanoHeaders { &self.headers }
       pub fn body(&self) -> Option<&Bytes> { self.body.as_ref() }
       
       pub fn set_body(&mut self, body: Bytes) {
           self.body = Some(body);
       }
       
       /// Convert to axum response (used in router handler)
       pub fn to_axum_response(&self) -> axum::response::Response<axum::body::Body> {
           let mut builder = axum::response::Response::builder()
               .status(self.status);
           
           // Add headers
           for (name, values) in self.headers.entries() {
               for value in values {
                   if let Ok(header_name) = axum::http::HeaderName::from_bytes(name.as_bytes()) {
                       if let Ok(header_value) = axum::http::HeaderValue::from_str(value) {
                           builder = builder.header(header_name, header_value);
                       }
                   }
               }
           }
           
           // Build response with body
           let body = self.body.clone()
               .map(|b| axum::body::Body::from(b))
               .unwrap_or_else(axum::body::Body::empty);
           
           builder.body(body).unwrap_or_default()
       }
       
       fn default_status_text(status: u16) -> String {
           match status {
               200 => "OK".to_string(),
               201 => "Created".to_string(),
               204 => "No Content".to_string(),
               400 => "Bad Request".to_string(),
               404 => "Not Found".to_string(),
               500 => "Internal Server Error".to_string(),
               _ => "Unknown".to_string(),
               }
       }
   }
   
   /// Builder pattern for NanoResponse
   impl NanoResponse {
       pub fn ok() -> Self {
           Self::new(200, NanoHeaders::default(), None)
       }
       
       pub fn with_body(mut self, body: impl Into<String>) -> Self {
           self.body = Some(Bytes::from(body.into()));
           self
       }
       
       pub fn with_header(mut self, name: &str, value: &str) -> Self {
           self.headers.set(name, value);
           self
       }
   }

   //! V8 Bridge for WinterTC types
   //!
   //! This module serializes Request/Response objects to V8 JavaScript contexts
   //! per D-06: JSON serialization → V8 parse (not direct V8 API creation).
   
   use crate::http::{NanoRequest, NanoResponse, NanoUrl, NanoHeaders};
   use crate::v8::isolate::NanoIsolate;
   use anyhow::{anyhow, Result};
   
   /// Serialize a NanoRequest to a JavaScript object in V8
   ///
   /// Per D-06, we serialize to JSON and parse in V8 rather than building
   /// objects directly with V8 API (simpler and less error-prone).
   pub fn request_to_v8(
       isolate: &mut NanoIsolate,
       request: &NanoRequest,
   ) -> Result<v8::Local<v8::Object>> {
       // Serialize request to JSON
       let request_json = serialize_request_to_json(request);
       
       // Create HandleScope
       let scope = &mut v8::HandleScope::new(isolate.isolate());
       
       // Parse JSON into V8 object
       let json_string = v8::String::new(scope, &request_json)
           .ok_or_else(|| anyhow!("Failed to create V8 string for request JSON"))?;
       
       // Use JSON.parse() to create the object
       let global = v8::Global::new(scope, json_string);
       let local = v8::Local::new(scope, global);
       
       // For now, return a placeholder object
       // Full implementation requires executing JSON.parse which needs a context
       // This will be completed in Phase 3 when we integrate with script execution
       let obj = v8::Object::new(scope);
       
       // Add basic properties
       let method_key = v8::String::new(scope, "method").unwrap();
       let method_val = v8::String::new(scope, request.method()).unwrap();
       obj.set(scope, method_key.into(), method_val.into());
       
       let url_key = v8::String::new(scope, "url").unwrap();
       let url_val = v8::String::new(scope, &request.url_string()).unwrap();
       obj.set(scope, url_key.into(), url_val.into());
       
       Ok(obj)
   }
   
   /// Serialize NanoRequest to JSON string
   fn serialize_request_to_json(request: &NanoRequest) -> String {
       // Build JSON manually to ensure correct WinterTC structure
       let mut json = String::from("{");
       
       // method
       json.push_str(&format!("\"method\":\"{}\",", escape_json(request.method())));
       
       // url
       json.push_str(&format!("\"url\":\"{}\",", escape_json(&request.url_string())));
       
       // headers
       json.push_str("\"headers\":{");
       let mut first = true;
       request.headers().for_each(|name, value| {
           if !first { json.push(','); }
           first = false;
           json.push_str(&format!("\"{}\":\"{}\"", 
               escape_json(name), 
               escape_json(value)));
       });
       json.push_str("},");
       
       // body (base64 encoded if present)
       if let Some(body) = request.body() {
           let base64 = base64::encode(body);
           json.push_str(&format!("\"body\":\"{}\",\"bodyUsed\":true", base64));
       } else {
           json.push_str("\"body\":null,\"bodyUsed\":false");
       }
       
       json.push('}');
       json
   }
   
   /// Parse JavaScript Response object returned from handler
   ///
   /// The JavaScript handler returns an object that we need to convert
   /// back to NanoResponse for HTTP response.
   pub fn response_from_v8(
       isolate: &mut NanoIsolate,
       value: v8::Local<v8::Value>,
   ) -> Result<NanoResponse> {
       let scope = &mut v8::HandleScope::new(isolate.isolate());
       
       // Check if it's an object
       if !value.is_object() {
           return Err(anyhow!("Handler return value is not an object"));
       }
       
       let obj = value.to_object(scope).ok_or_else(|| anyhow!("Failed to convert to object"))?;
       
       // Extract status
       let status_key = v8::String::new(scope, "status").unwrap();
       let status_val = obj.get(scope, status_key.into())
           .ok_or_else(|| anyhow!("Response missing status"))?;
       let status = status_val.to_integer(scope)
           .map(|i| i.value() as u16)
           .unwrap_or(200);
       
       // Extract headers (if present)
       let mut headers = NanoHeaders::new();
       let headers_key = v8::String::new(scope, "headers").unwrap();
       if let Some(headers_val) = obj.get(scope, headers_key.into()) {
           if headers_val.is_object() {
               // TODO: Parse headers object
               // For now, use default empty headers
           }
       }
       
       // Extract body
       let mut body = None;
       let body_key = v8::String::new(scope, "body").unwrap();
       if let Some(body_val) = obj.get(scope, body_key.into()) {
           if !body_val.is_null_or_undefined() {
               if let Some(body_str) = body_val.to_string(scope) {
                   let body_text = body_str.to_rust_string_lossy(scope);
                   body = Some(bytes::Bytes::from(body_text));
               }
           }
       }
       
       Ok(NanoResponse::new(status, headers, body))
   }
   
   /// Escape string for JSON safety
   fn escape_json(s: &str) -> String {
       s.replace('\\', "\\\\")
        .replace('"', "\\\"")
        .replace('\n', "\\n")
        .replace('\r', "\\r")
        .replace('\t', "\\t")
   }
