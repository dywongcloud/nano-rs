<task type="auto">
  <name>Task 5: Add integration test for server startup</name>
  <files>tests/http_server_test.rs</files>
  <action>
Create `tests/http_server_test.rs` with an integration test that starts the server and makes an HTTP request:

```rust
//! HTTP server integration tests

use nano::http::{start_server, ServerConfig};
use std::time::Duration;
use tokio::time::sleep;

#[tokio::test]
async fn test_server_starts_and_responds() {
    // Start server on random port
    let config = ServerConfig {
        port: 0, // Let OS assign port
        host: "127.0.0.1".to_string(),
    };
    
    // Spawn server in background
    tokio::spawn(async move {
        start_server(config).await.expect("Server failed to start");
    });
    
    // Give server time to start
    sleep(Duration::from_millis(100)).await;
    
    // Make request to health endpoint
    let response = reqwest::get("http://127.0.0.1:8080/health")
        .await
        .expect("Request failed");
    
    assert_eq!(response.status(), 200);
}
```

Note: This requires adding `reqwest` as a dev-dependency in Cargo.toml:
```toml
[dev-dependencies]
reqwest = { version = "0.12", features = ["rustls-tls"] }
```

The test verifies:
