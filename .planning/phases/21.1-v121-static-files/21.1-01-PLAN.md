# Plan 21.1-01: Auto-Detect Entrypoint Type and Serve Static Files

## Goal
Implement automatic entrypoint type detection where:
- `.js`, `.mjs`, `.ts` files → Execute as JavaScript Worker
- All other files (`.html`, `.css`, `.png`, etc.) → Serve statically

## Background
Currently, NANO expects all entrypoints to be JavaScript files. When users point to an HTML file (like Astro's `index.html`), it fails with `SyntaxError: Unexpected token '<'` because NANO tries to parse HTML as JavaScript.

## Requirements
- Detect file extension of entrypoint
- For JS files: Execute as Worker (current behavior)
- For non-JS files: Serve statically with correct content-type
- Support directory entrypoints (serve `index.html` for `/`)

## Implementation

### 1. Entrypoint Detection Logic
**File:** `src/http/router.rs` or `src/http/server.rs`

```rust
pub enum EntrypointType {
    JavaScript(String),  // Path to JS file
    StaticFile(String),  // Path to specific file
    StaticDir(String),   // Path to directory (serve index.html)
}

fn detect_entrypoint_type(path: &str) -> EntrypointType {
    let ext = Path::new(path).extension()
        .and_then(|e| e.to_str())
        .unwrap_or("");
    
    match ext.to_lowercase().as_str() {
        "js" | "mjs" | "ts" => EntrypointType::JavaScript(path.to_string()),
        "html" => EntrypointType::StaticFile(path.to_string()),
        _ => {
            // Check if it's a directory
            if Path::new(path).is_dir() {
                EntrypointType::StaticDir(path.to_string())
            } else {
                EntrypointType::StaticFile(path.to_string())
            }
        }
    }
}
```

### 2. Static File Handler
**File:** `src/http/router.rs`

Add new handler type:
```rust
pub enum HandlerType {
    // ... existing variants
    StaticFile { path: String, content_type: String },
    StaticDir { root: String },
}
```

Implement static file serving:
```rust
impl RouteTarget {
    pub async fn handle(&self, request: NanoRequest) -> NanoResponse {
        match &self.handler_type {
            // ... existing handlers
            HandlerType::StaticFile { path, content_type } => {
                // Read file from filesystem
                match tokio::fs::read_to_string(path).await {
                    Ok(content) => NanoResponse::ok()
                        .with_header("Content-Type", content_type)
                        .with_body(content),
                    Err(_) => NanoResponse::not_found(),
                }
            }
            HandlerType::StaticDir { root } => {
                let path = request.url().pathname();
                let file_path = if path == "/" {
                    format!("{}/index.html", root)
                } else {
                    format!("{}{}", root, path)
                };
                
                // Determine content type from extension
                let ext = Path::new(&file_path).extension()
                    .and_then(|e| e.to_str())
                    .unwrap_or("");
                let content_type = match ext {
                    "html" => "text/html",
                    "css" => "text/css",
                    "js" => "application/javascript",
                    "json" => "application/json",
                    "png" => "image/png",
                    "jpg" | "jpeg" => "image/jpeg",
                    "svg" => "image/svg+xml",
                    _ => "application/octet-stream",
                };
                
                match tokio::fs::read(&file_path).await {
                    Ok(bytes) => NanoResponse::ok()
                        .with_header("Content-Type", content_type)
                        .with_body_bytes(bytes),
                    Err(_) => NanoResponse::not_found(),
                }
            }
        }
    }
}
```

pub fn content_type_from_ext(ext: &str) -> &'static str {
    match ext.to_lowercase().as_str() {
        "html" | "htm" => "text/html",
        "css" => "text/css",
        "js" | "mjs" => "application/javascript",
        "json" => "application/json",
        "xml" => "application/xml",
        "png" => "image/png",
        "jpg" | "jpeg" => "image/jpeg",
        "gif" => "image/gif",
        "svg" => "image/svg+xml",
        "ico" => "image/x-icon",
        "woff" => "font/woff",
        "woff2" => "font/woff2",
        "ttf" => "font/ttf",
        "otf" => "font/otf",
        "eot" => "application/vnd.ms-fontobject",
        "pdf" => "application/pdf",
        "zip" => "application/zip",
        "gz" => "application/gzip",
        "txt" => "text/plain",
        "md" => "text/markdown",

    }
