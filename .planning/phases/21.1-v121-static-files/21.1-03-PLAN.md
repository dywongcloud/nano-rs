# Plan 21.1-03: Sliver Creation from Directory (No Running App Required)

## Goal
Enable sliver creation directly from a directory without requiring the app to be running first. The sliver should pack all app files and be runnable standalone.

## Background
Currently, `sliver create` requires a running app to snapshot. Users want to create slivers from static builds (Astro, Next.js) without starting a server first.

## Requirements
- Create sliver from directory: `nano-rs sliver create --from-dir ./dist --name myapp`
- Pack all files (not just JS) into sliver
- Sliver runs standalone without external file dependencies
- Preserve directory structure in sliver

## Implementation

### 1. New CLI Command
**File:** `src/cli/commands/sliver.rs`

Add `--from-dir` option:
```rust
#[derive(Args)]
pub struct SliverCreateArgs {
    #[arg(help = "Hostname (legacy, optional when using --from-dir)")]
    pub hostname: Option<String>,
    
    #[arg(short, long, help = "Sliver name")]
    pub name: String,
    
    #[arg(short, long, help = "Tag/version")]
    pub tag: Option<String>,
    
    #[arg(short, long, help = "Output file path")]
    pub output: Option<String>,
    
    #[arg(long, help = "Create from directory (no running app needed)")]
    pub from_dir: Option<String>,
}

pub async fn execute(args: SliverCreateArgs) -> Result<()> {
    if let Some(dir) = args.from_dir {
        // Create from directory
        create_sliver_from_directory(&dir, &args.name, args.tag, args.output).await
    } else if let Some(hostname) = args.hostname {
        // Legacy: create from running app
        create_sliver_from_running_app(&hostname, &args.name, args.tag, args.output).await
    } else {
        bail!("Either --from-dir or hostname must be specified")
    }
}
```

### 2. Directory to Sliver Creation
**File:** `src/sliver/packager.rs` (new file)

```rust
use std::path::Path;
use tar::Builder;
use flate2::write::GzEncoder;
use flate2::Compression;

pub async fn create_sliver_from_directory(
    source_dir: &str,
    name: &str,
    tag: Option<String>,
    output: Option<String>,
) -> Result<()> {
    let output_path = output.unwrap_or_else(|| format!("{}.sliver", name));
    let source_path = Path::new(source_dir);
    
    // Create tarball
    let tar_gz = File::create(&output_path)?;
    let enc = GzEncoder::new(tar_gz, Compression::default());
    let mut tar = Builder::new(enc);
    
    // Add all files from directory
    tar.append_dir_all("files", source_path)?;
    
    // Add metadata
    let metadata = SliverMetadata {
        name: name.to_string(),
        tag: tag.unwrap_or_else(|| "latest".to_string()),
        created_at: chrono::Utc::now(),
        entrypoint: detect_entrypoint(source_path),
    };
    
    let metadata_json = serde_json::to_string(&metadata)?;
    let mut header = tar::Header::new_gnu();
    header.set_path("sliver.json")?;
    header.set_size(metadata_json.len() as u64);
    header.set_mode(0o644);
    header.set_cksum();
    tar.append(&header, metadata_json.as_bytes())?;
    
    tar.finish()?;
    
    println!("Created sliver: {}", output_path);
    Ok(())
}

fn detect_entrypoint(dir: &Path) -> String {
    // Check for common entrypoints
    let candidates = vec![
        "index.js",
        "index.mjs",
        "main.js",
        "worker.js",
        "index.html",  // For static sites
    ];
    
    for candidate in candidates {
        if dir.join(candidate).exists() {
            return format!("/files/{}", candidate);
        }
    }
    
    // Default to index.html if it exists
    if dir.join("index.html").exists() {
        return "/files/index.html".to_string();
    }
    
    panic!("No entrypoint found in directory");
}
```

### 3. Sliver Runtime with Embedded Files
**File:** `src/sliver/runtime.rs`

When running a sliver created from directory:
```rust
pub async fn run_sliver(sliver_path: &str) -> Result<()> {
    // Extract sliver
    let temp_dir = tempdir()?;
    extract_sliver(sliver_path, temp_dir.path()).await?;
    
    // Read metadata
    let metadata: SliverMetadata = read_metadata(temp_dir.path()).await?;
    
    // Load files into VFS
    let vfs = IsolateVfs::new();
    let files_dir = temp_dir.path().join("files");
    
    if files_dir.exists() {
        load_directory_to_vfs(&vfs, files_dir.to_str().unwrap(), "/").await?;
    }
    
    // Start server with VFS
    let entrypoint = temp_dir.path().join(&metadata.entrypoint.trim_start_matches('/'));
    
    if metadata.entrypoint.ends_with(".html") {
        // Static site: serve HTML directly
        start_static_server(vfs, &metadata.entrypoint).await?;
    } else {
        // JS Worker: execute
        start_worker_server(vfs, &entrypoint).await?;
    }
    
    Ok(())
}
```

## Usage Examples

### Create from Astro build:
```bash
# Build Astro
npm run build

# Create sliver from dist
nano-rs sliver create --from-dir ./dist --name astroblog --tag v1.0

# Run sliver
nano-rs run --sliver astroblog.sliver
```

### Create from any directory:
```bash
nano-rs sliver create --from-dir ./my-static-site --name mysite
```
