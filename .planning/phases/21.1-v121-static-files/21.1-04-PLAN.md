# Plan 21.1-04: Test Suite for Static File Serving

## Goal
Create comprehensive tests for static file serving functionality to ensure v1.2.1 features work correctly and maintain backward compatibility.

## Requirements
- Test static HTML serving
- Test directory serving (index.html fallback)
- Test content-type detection
- Test 404 handling
- Test sliver creation from directory
- Regression tests for JS entrypoints

### 2. Test Cases

**Test 1: HTML Entrypoint**
```javascript
await runTest('Static: HTML entrypoint serves file', async () => {
  const configPath = createConfig([{
    hostname: 'static.local',
    entrypoint: './test-apps/static-file-test/dist/index.html'
  }], port);
  
  const nano = startNanoServer(configPath, port);
  await waitForServer(port, 10000);
  
  const response = await httpRequest({
    hostname: 'static.local',
    port,
    path: '/',
    method: 'GET'
  });
  
  assertEquals(response.status, 200, 'Should return 200');
  assertContains(response.body, 'Static File Test', 'Should contain HTML content');
  assertEquals(response.headers['content-type'], 'text/html', 'Should have correct content-type');
  
  stopNano(nano);
}, 'static');
```

**Test 6: Sliver from Directory**
```javascript
await runTest('Static: Create sliver from directory', async () => {
  const output = execSync(
    `${CONFIG.NANO_BINARY} sliver create --from-dir ./test-apps/static-file-test/dist --name static-test --tag v1.0`,

  assertContains(output.toLowerCase(), 'created', 'Should confirm sliver creation');

  const sliverPath = path.join(process.cwd(), 'static-test.sliver');

  // Cleanup

```

**Test 7: Run Sliver Standalone**
```javascript
await runTest('Static: Run sliver without source directory', async () => {
  // First create sliver
  execSync(
    `${CONFIG.NANO_BINARY} sliver create --from-dir ./test-apps/static-file-test/dist --name static-run-test`,
    { encoding: 'utf8', timeout: 30000 }
  );
  
  const sliverPath = path.join(process.cwd(), 'static-run-test.sliver');
  
  // Copy sliver to temp location
  const tempSliver = path.join('/tmp', 'static-run-test.sliver');
  fs.copyFileSync(sliverPath, tempSliver);
  fs.unlinkSync(sliverPath);  // Remove original
  
  // Run from temp location (no access to source)
  const nano = spawn(CONFIG.NANO_BINARY, ['run', '--sliver', tempSliver, '--port', port.toString()]);
  await waitForServer(port, 10000);
  
  const response = await httpRequest({
    hostname: 'localhost',
    port,
    path: '/',
    method: 'GET'
  });
  
  assertEquals(response.status, 200, 'Should serve from sliver');
  assertContains(response.body, 'Static File Test', 'Should have correct content');
  
  nano.kill('SIGTERM');
  fs.unlinkSync(tempSliver);
}, 'static');
```
