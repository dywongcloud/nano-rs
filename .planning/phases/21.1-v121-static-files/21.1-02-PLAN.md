# Plan 21.1-02: VFS Directory Loading for Static Assets

## Goal
Implement VFS (Virtual File System) directory loading so that static assets can be loaded into memory and served efficiently. This is especially important for slivers and multi-file apps.

## Background
The VFS exists but currently requires manual file-by-file operations. We need the ability to load an entire directory tree into VFS at startup.

## Requirements
- Load all files from a directory into VFS recursively
- Preserve directory structure in VFS paths
- Support binary files (not just text)
- Efficient memory usage for large directories

## Implementation

### 1. VFS Directory Loader
**File:** `src/vfs/loader.rs` (new file)

```rust
use std::path::Path;
use tokio::fs;
use crate::vfs::{IsolateVfs, VfsError};

pub async fn load_directory_to_vfs(
    vfs: &IsolateVfs,
    source_dir: &str,
    mount_point: &str,
) -> Result<usize, VfsError> {
    let mut count = 0;
    let source_path = Path::new(source_dir);
    
    let mut entries = fs::read_dir(source_path).await
        .map_err(|e| VfsError::IoError(e.to_string()))?;
    
    while let Some(entry) = entries.next_entry().await
        .map_err(|e| VfsError::IoError(e.to_string()))? 
    {
        let path = entry.path();
        let file_name = entry.file_name();
        let file_name_str = file_name.to_string_lossy();
        
        let vfs_path = if mount_point.ends_with('/') {
            format!("{}{}", mount_point, file_name_str)
        } else {
            format!("/{}/{}", mount_point, file_name_str)
        };
        
        if path.is_dir() {
            // Recursively load subdirectory
            let sub_count = load_directory_to_vfs(
                vfs,
                path.to_str().unwrap(),
                &vfs_path,
            ).await?;
            count += sub_count;
        } else {
            // Load file
            let content = fs::read(&path).await
                .map_err(|e| VfsError::IoError(e.to_string()))?;
            
            vfs.write(&vfs_path, &content).await?;
            count += 1;
        }
    }
    
    Ok(count)
}
```

## Testing

### Test 1: Load Directory
```rust
#[tokio::test]
async fn test_load_directory() {
    let vfs = IsolateVfs::new();
    let count = load_directory_to_vfs(&vfs, "./test-assets", "/dist").await.unwrap();
    
    assert!(count > 0);
    
    // Verify files are accessible
    let content = vfs.read("/dist/index.html").await.unwrap();
    assert!(!content.is_empty());
}
```
