## What Was Built

### 1. Entrypoint Type Detection (`src/http/router.rs`)

**New `EntrypointType` enum:**
```rust
pub enum EntrypointType {
    JavaScript(String),  // Execute as JS Worker
    StaticFile(String),  // Serve specific file
    StaticDir(String),   // Serve directory with index.html fallback
}
```

**Detection logic:**
- Checks if path is a directory → `StaticDir`
- Checks file extension:
  - `.js`, `.mjs`, `.ts` → `JavaScript`
  - All other extensions → `StaticFile`

### 2. New Handler Types (`src/http/router.rs`)

**Added to `HandlerType` enum:**
```rust
StaticFile {
    path: String,
    content_type: String,
}
StaticDir {
    root: String,
    default_file: String,
}
```

**Implementation:**
- `StaticFile`: Reads file from filesystem, serves with correct content-type
- `StaticDir`: Serves files from directory, with index.html fallback for root path
- Both include security checks (path traversal prevention)

### 3. Content-Type Mapping (`src/http/types.rs`)

**New `content_type_from_ext()` function** with 30+ MIME type mappings:

| Category | Extensions | MIME Type |
|----------|-----------|-----------|
| HTML | html, htm | text/html; charset=utf-8 |
| CSS | css | text/css; charset=utf-8 |
| JavaScript | js, mjs | application/javascript; charset=utf-8 |
| JSON | json | application/json; charset=utf-8 |
| Images | png, jpg, gif, svg, webp, ico | image/* |
| Fonts | woff2, woff, ttf, otf | font/* |
| Archives | zip, gz, tar | application/* |
| Media | mp4, mp3, webm | video/*, audio/* |

### 4. Config Integration (`src/http/server.rs`)

**Updated `start_server_with_config()`** to use entrypoint detection:

For each app in config:
1. Detect entrypoint type using `detect_entrypoint_type()`
2. Create appropriate handler based on type:
   - JavaScript → `WinterCGHandler` or `WinterCGSliverHandler`
   - StaticFile → `StaticFile` handler with content-type
   - StaticDir → `StaticDir` handler with index.html fallback
3. Log the detected handler type for visibility

### 5. Dispatch Integration (`src/http/router.rs`)

**Updated `dispatch_to_worker_pool()`** to handle static handlers:

```rust
HandlerType::StaticFile { .. } | HandlerType::StaticDir { .. } => {
    // Serve directly without JS worker dispatch
    let nano_response = target.handle(nano_request).await;
    return nano_response.to_axum_response();
}
```
