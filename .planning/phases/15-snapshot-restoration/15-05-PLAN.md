# Phase 15 Plan 05: Performance Benchmarks and Migration Tests

## Objective
Create comprehensive performance benchmarks to verify the ~1-2ms cold start target and test sliver portability between NANO instances.

## Context

### Performance Targets
- Cold start from sliver: ~1-2ms
- Compare: context reset ~5ms, new isolate ~50-100ms

### Testing Requirements
- Unit tests for unpacker integration
- Integration test: pack -> unpack -> verify identical
- Performance benchmark: measure cold start time
- Migration test: create on instance A, restore on instance B

### Current State
- Sliver packing exists (Phase 14)
- Sliver unpacking exists
- VFS restoration exists
- V8 snapshot restoration (placeholder in v135)

### Task 2: Create migration integration test

**Type:** auto

**Action:**
Create test file `tests/sliver_migration.rs`:

Test scenarios:
1. Pack on "instance A", unpack on "instance B"
2. Verify metadata matches
3. Verify heap data is identical
4. Verify VFS entries match
5. Test with different VFS backends (memory, disk)

The test simulates migration by:
- Creating sliver with known content
- Serializing to bytes (simulates transfer)
- Deserializing from bytes
- Verifying all content matches

**Acceptance Criteria:**
- Test creates sliver with known content
- Test simulates transfer (bytes serialization)
- Test verifies metadata matches
- Test verifies heap data identical
- Test verifies VFS entries match
- Test passes: cargo test --test sliver_migration
