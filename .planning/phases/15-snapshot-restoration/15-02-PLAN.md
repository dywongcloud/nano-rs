# Phase 15 Plan 02: V8 Snapshot Restoration

## Objective
Implement V8 heap snapshot loading to restore isolates from snapshot blobs. Create the `restore_from_snapshot()` function that initializes a NanoIsolate from a heap.bin blob.

## Context

### Current State
The `src/v8/snapshot.rs` module has placeholder snapshot creation (returns marker blob). V8's snapshot restoration API is available through `v8::Isolate::new_with_snapshot()` when rusty_v8 exposes it.

### V8 Snapshot API
- `v8::Isolate::new_with_snapshot(params, snapshot_blob)` — Creates isolate from snapshot
- Snapshot blob must be valid V8 heap snapshot data
- External reference table must be handled during restoration

### Important Note
Due to rusty_v8 v135 API limitations, full heap snapshots may not be fully supported. The implementation should:
1. Provide the API for snapshot restoration
2. Handle the placeholder case gracefully (revert to fresh isolate if snapshot is marker)
3. Be ready for full implementation when V8 API improves

```rust
/// Restore an isolate from a heap snapshot blob
///
/// This function attempts to create a new isolate initialized with the
/// provided snapshot data. If the snapshot is a placeholder or invalid,
/// it returns an error.
///
/// # Arguments
/// * `snapshot_data` - The V8 heap snapshot blob (from heap.bin)
/// * `vfs` - The IsolateVfs to attach to the restored isolate
///
/// # Returns
/// A NanoIsolate restored from snapshot, or SnapshotError if restoration fails
pub fn restore_from_snapshot(
    snapshot_data: &[u8],
    vfs: crate::vfs::IsolateVfs,
) -> SnapshotResult<crate::v8::isolate::NanoIsolate> {
    // Check for placeholder
    if is_placeholder_snapshot(snapshot_data) {
        return Err(SnapshotError::InvalidIsolateState(
            "Cannot restore from placeholder snapshot".to_string()
        ));
    }

    // Validate snapshot data is non-empty
    if snapshot_data.is_empty() {
        return Err(SnapshotError::InvalidIsolateState(
            "Snapshot data is empty".to_string()
        ));
    }

    // Create isolate from snapshot using V8 API
    // Note: In v135, snapshot restoration may use startup snapshot API
    match create_isolate_from_snapshot(snapshot_data, vfs) {
        Ok(isolate) => {
            tracing::info!("Restored isolate from snapshot ({} bytes)", snapshot_data.len());
            Ok(isolate)
        }
        Err(e) => {
            tracing::error!("Failed to restore isolate from snapshot: {}", e);
            Err(SnapshotError::CreationFailed(e.to_string()))
        }
    }
}

/// Check if snapshot data is a placeholder
fn is_placeholder_snapshot(data: &[u8]) -> bool {
    data == b"NANO_SNAPSHOT_PLACEHOLDER_V1"
}

/// Create isolate from snapshot data
fn create_isolate_from_snapshot(
    snapshot_data: &[u8],
    vfs: crate::vfs::IsolateVfs,
) -> anyhow::Result<crate::v8::isolate::NanoIsolate> {
    // Use NanoIsolate's constructor that supports snapshot data
    // This delegates to the isolate module's implementation
    crate::v8::isolate::NanoIsolate::from_snapshot(snapshot_data, vfs)
}
```

### Task 2: Add from_snapshot constructor to NanoIsolate

**Type:** auto

```rust
impl NanoIsolate {
    /// Create a new V8 isolate from a snapshot blob
    ///
    /// This is the primary constructor for restoring isolates from
    /// sliver snapshots. The snapshot contains the serialized V8 heap state.
    ///
    /// # Arguments
    /// * `snapshot_data` - The V8 heap snapshot blob
    /// * `vfs` - The VFS configuration for this isolate
    ///
    /// # Platform Requirement
    /// The V8 platform MUST be initialized before calling this function.
    pub fn from_snapshot(
        snapshot_data: &[u8],
        vfs: IsolateVfs,
    ) -> Result<Self> {
        // Create snapshot blob reference
        let snapshot_blob = v8::SnapshotBlob {
            data: snapshot_data,
            raw_size: snapshot_data.len(),
            // TODO: Handle external references count if needed
        };

        // Create params with snapshot
        let params = v8::CreateParams::default()
            .snapshot_blob(&snapshot_blob);

        // Create isolate with snapshot
        let mut isolate = v8::Isolate::new(params);

        // Create the EPT fix sentinel (same as new())
        let sentinel = {
            let scope = &mut v8::HandleScope::new(&mut isolate);
            let undefined = v8::undefined(scope);
            let value: v8::Local<v8::Value> = undefined.into();
            v8::Global::new(scope, value)
        };

        tracing::info!(
            "Created NanoIsolate from snapshot ({} bytes)",
            snapshot_data.len()
        );

        Ok(Self {
            sentinel,
            isolate,
            _not_send_sync: PhantomData,
            vfs,
        })
    }
}
```

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use crate::v8::platform;
    use crate::vfs::{IsolateVfs, MemoryBackend, VfsNamespace};
    use std::sync::Arc;

    fn init_platform() {
        platform::initialize_platform().expect("Failed to init platform");
    }

    #[test]
    fn test_restore_from_snapshot_rejects_placeholder() {
        init_platform();
        
        let placeholder = b"NANO_SNAPSHOT_PLACEHOLDER_V1";
        let vfs = IsolateVfs::new(
            VfsNamespace::from_hostname("test"),
            Arc::new(MemoryBackend::default()),
        );
        
        let result = restore_from_snapshot(placeholder, vfs);
        assert!(result.is_err());
        assert!(matches!(result.unwrap_err(), SnapshotError::InvalidIsolateState { .. }));
    }

    #[test]
    fn test_restore_from_snapshot_rejects_empty() {
        init_platform();
        
        let vfs = IsolateVfs::new(
            VfsNamespace::from_hostname("test"),
            Arc::new(MemoryBackend::default()),
        );
        
        let result = restore_from_snapshot(&[], vfs);
        assert!(result.is_err());
    }

    #[test]
    fn test_is_placeholder_detection() {
        assert!(is_placeholder_snapshot(b"NANO_SNAPSHOT_PLACEHOLDER_V1"));
        assert!(!is_placeholder_snapshot(b"real data"));
        assert!(!is_placeholder_snapshot(&[]));
    }

    #[test]
    fn test_from_snapshot_creates_isolate() {
        init_platform();
        
        // Use valid-looking snapshot data (V8 will handle it)
        let snapshot_data = vec![0u8; 1024];
        let vfs = IsolateVfs::new(
            VfsNamespace::from_hostname("test"),
            Arc::new(MemoryBackend::default()),
        );
        
        // This may succeed or fail depending on V8 version,
        // but should not panic
        let _result = NanoIsolate::from_snapshot(&snapshot_data, vfs);
    }

    #[test]
    fn test_restored_isolate_has_vfs() {
        init_platform();
        
        let snapshot_data = vec![0u8; 1024];
        let vfs = IsolateVfs::new(
            VfsNamespace::from_hostname("test.example.com"),
            Arc::new(MemoryBackend::default()),
        );
        
        if let Ok(isolate) = NanoIsolate::from_snapshot(&snapshot_data, vfs) {
            assert_eq!(isolate.vfs().namespace().as_str(), "test.example.com");
        }
    }
}
```
