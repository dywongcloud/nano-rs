# Phase 17: Request/Response Fixes

**Phase:** 17-request-response-fixes  
**Type:** Remediation  
**Goal:** Fix Bug #1 (Incomplete Request) and Bug #4 (Async Support) from blackbox evaluation  
**Success Criteria:** Full WinterCG Request object passed to JS; async handlers with `await` resolve correctly; request body readable as text/JSON

---

## Problem Summary

### Bug #1: Incomplete Request Object

**Current State:**
- Lines 85, 135 in `src/worker/pool.rs` create a minimal request:
  ```rust
  let request_json = format!("{{\"method\":\"{}\"}}", handler_ctx.request.method());
  ```
- Only the method string is passed to JavaScript handlers
- URL, headers, and body are lost

**Impact:**
- Handlers cannot access the full request URL (path, query params)
- Headers (like `Content-Type`, `Authorization`) are unavailable
- POST/PUT bodies cannot be read
- Frameworks like Hono.js that inspect the URL fail

### Bug #4: Async/Promise Support Missing

**Current State:**
- `execute_handler_in_context()` and `execute_handler_code()` call the JS handler synchronously
- `scope.perform_microtask_checkpoint()` is never called
- Promises returned by async handlers remain in Pending state

**Impact:**
- `async function fetch(request) { ... }` never resolves
- `await fetch()` inside handlers fails
- Modern JavaScript patterns are unsupported

---

## Decision: Bridge Implementation to pool.rs

**D-17-01:** Use `src/runtime/handler.rs` patterns in `pool.rs`

The `execute_handler_with_context()` function in `src/runtime/handler.rs` already:
- Serializes full Request objects using `serialize_request_to_json()`
- Parses JSON into a proper JS object via `JSON.parse()`
- Has Promise handling code (lines 288-316)

**Approach:** Adapt these patterns into the worker pool execution flow.

   ```rust
   //! Request object helpers for WinterCG compatibility
   
   use v8;
   
   /// Bind Request prototype methods (text, json, arrayBuffer, blob)
   pub fn bind_request_api(
       scope: &mut v8::ContextScope<v8::HandleScope>,
       context: v8::Local<v8::Context>,
   ) {
       // Get or create Request constructor
       let global = context.global(scope);
       let request_key = v8::String::new(scope, "Request").unwrap();
       
       // If Request exists, add methods to prototype
       if let Some(request_ctor) = global.get(scope, request_key.into()) {
           if let Some(request_obj) = request_ctor.to_object(scope) {
               let prototype_key = v8::String::new(scope, "prototype").unwrap();
               if let Some(prototype) = request_obj.get(scope, prototype_key.into()) {
                   if let Some(proto_obj) = prototype.to_object(scope) {
                       bind_request_method(scope, proto_obj, "text", request_text_callback);
                       bind_request_method(scope, proto_obj, "json", request_json_callback);
                       bind_request_method(scope, proto_obj, "arrayBuffer", request_arraybuffer_callback);
                   }
               }
           }
       }
   }
   
   fn bind_request_method(
       scope: &mut v8::ContextScope<v8::HandleScope>,
       prototype: v8::Local<v8::Object>,
       name: &str,
       callback: impl v8::MapFnTo<v8::FunctionCallback>,
   ) {
       let name = v8::String::new(scope, name).unwrap();
       let func = v8::Function::new(scope, callback).unwrap();
       prototype.set(scope, name.into(), func.into());
   }
   
   fn request_text_callback(
       scope: &mut v8::HandleScope,
       args: v8::FunctionCallbackArguments,
       mut retval: v8::ReturnValue,
   ) {
       // Get 'this' (the request object)
       let this = args.this();
       
       // Extract body from the request object (base64 encoded in body property)
       let body_key = v8::String::new(scope, "body").unwrap();
       if let Some(body_val) = this.get(scope, body_key.into()) {
           if !body_val.is_null() && !body_val.is_undefined() {
               if let Some(body_str) = body_val.to_string(scope) {
                   let base64_body = body_str.to_rust_string_lossy(scope);
                   // Decode base64 and return as string
                   if let Ok(decoded) = base64_decode(&base64_body) {
                       let text = String::from_utf8_lossy(&decoded);
                       let result = v8::String::new(scope, &text).unwrap();
                       retval.set(result.into());
                       return;
                   }
               }
           }
       }
       
       // Return empty string if no body
       let empty = v8::String::new(scope, "").unwrap();
       retval.set(empty.into());
   }
   
   fn request_json_callback(
       scope: &mut v8::HandleScope,
       args: v8::FunctionCallbackArguments,
       mut retval: v8::ReturnValue,
   ) {
       // Get text first, then JSON.parse it
       let this = args.this();
       let body_key = v8::String::new(scope, "body").unwrap();
       
       if let Some(body_val) = this.get(scope, body_key.into()) {
           if !body_val.is_null() && !body_val.is_undefined() {
               if let Some(body_str) = body_val.to_string(scope) {
                   let base64_body = body_str.to_rust_string_lossy(scope);
                   if let Ok(decoded) = base64_decode(&base64_body) {
                       let text = String::from_utf8_lossy(&decoded);
                       
                       // Parse JSON
                       let global = scope.get_current_context().global(scope);
                       let json_key = v8::String::new(scope, "JSON").unwrap();
                       if let Some(json_val) = global.get(scope, json_key.into()) {
                           if let Some(json_obj) = json_val.to_object(scope) {
                               let parse_key = v8::String::new(scope, "parse").unwrap();
                               if let Some(parse_fn) = json_obj.get(scope, parse_key.into()) {
                                   if let Some(parse) = parse_fn.cast::<v8::Function>() {
                                       let text_str = v8::String::new(scope, &text).unwrap();
                                       if let Some(result) = parse.call(scope, json_val.into(), &[text_str.into()]) {
                                           retval.set(result);
                                           return;
                                       }
                                   }
                               }
                           }
                       }
                   }
               }
           }
       }
       
       // Return null on error
       retval.set(v8::null(scope).into());
   }
   
   fn request_arraybuffer_callback(
       scope: &mut v8::HandleScope,
       args: v8::FunctionCallbackArguments,
       mut retval: v8::ReturnValue,
   ) {
       let this = args.this();
       let body_key = v8::String::new(scope, "body").unwrap();
       
       if let Some(body_val) = this.get(scope, body_key.into()) {
           if !body_val.is_null() && !body_val.is_undefined() {
               if let Some(body_str) = body_val.to_string(scope) {
                   let base64_body = body_str.to_rust_string_lossy(scope);
                   if let Ok(decoded) = base64_decode(&base64_body) {
                       // Create ArrayBuffer from decoded bytes
                       let buffer = v8::ArrayBuffer::new(scope, decoded.len());
                       if let SomeBackingStore(buffer_backing) = buffer.get_backing_store() {
                           let data = buffer_backing.data();
                           unsafe {
                               std::ptr::copy_nonoverlapping(
                                   decoded.as_ptr(),
                                   data as *mut u8,
                                   decoded.len()
                               );
                           }
                           retval.set(buffer.into());
                           return;
                       }
                   }
               }
           }
       }
       
       // Return empty ArrayBuffer if no body
       let empty = v8::ArrayBuffer::new(scope, 0);
       retval.set(empty.into());
   }
   
   fn base64_decode(input: &str) -> Result<Vec<u8>, ()> {
       // Implementation of base64 decoding
       use base64::Engine;
       base64::engine::general_purpose::STANDARD
           .decode(input)
           .map_err(|_| ())
   }
   ```

## Threat Model

### Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP Request → V8 | Untrusted input crosses into JavaScript execution |
| JS Handler → Response | Handler output determines HTTP response |

### STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-01 | Injection | Request body deserialization | Mitigate | JSON.parse() errors are caught; malformed body returns 400 |
| T-17-02 | Denial of Service | Promise resolution | Mitigate | Add timeout to Promise resolution (5s default, configurable) |
| T-17-03 | Information Disclosure | Error messages | Mitigate | Internal V8 errors sanitized; Promise rejections logged but sanitized in response |
| T-17-04 | Tampering | Base64 body encoding | Accept | Base64 is internal encoding; JS side decodes. Integrity not critical for this phase. |
