<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JS → Rust fetch() | Untrusted JavaScript initiates outbound requests |
| HTTP client → External network | Outbound connections leave the runtime |
| Response stream → JS | External data flows into isolate |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01 | Spoofing | URL validation | mitigate | Validate URLs are absolute (no javascript: scheme), reject file://, ftp:// |
| T-06-02 | Tampering | Request headers | mitigate | Filter dangerous headers (Host, Content-Length when chunked), validate header values |
| T-06-03 | DoS | Connection pool | mitigate | Limit max concurrent connections per isolate (100), enforce request timeout |
| T-06-04 | DoS | Response size | mitigate | Max response size limit (100MB default), abort on limit exceeded |
| T-06-05 | Information Disclosure | Redirect handling | mitigate | Limit redirects (10 hops), validate redirect URLs don't point to internal services |
| T-06-06 | DoS | Memory exhaustion | mitigate | Bounded channel between hyper and V8 (backpressure), stream large bodies |
| T-06-07 | Elevation | SSRF prevention | mitigate | Block private IP ranges (10.0.0.0/8, 192.168.0.0/16, etc.) via URL filter |
</threat_model>

```bash
async function fetch(request) {
  // Make outbound request
  const response = await fetch('https://httpbin.org/get');
  console.log('Status:', response.status);
  
  // Stream the body
  const reader = response.body.getReader();
  let chunks = [];
  while (true) {
    const {done, value} = await reader.read();
    if (done) break;
    chunks.push(value);
  }
  
  const text = new TextDecoder().decode(Buffer.concat(chunks));
  return new Response(text);
}
EOF
```
