Approach 6 summary to add:
  Status: WORKING (as of commit 3f098832)
  Pattern: Persistent HandleScope + ContextScope on the thread stack, never dropped
    between requests within one isolate lifetime.
  Handler cache: Global<Function> cached per entrypoint, converted to Local via
    v8::Local::new(&mut ctx_scope, handler_g) in the SAME ContextScope.
  Required conditions for correctness:
    1. TryCatch RAII wrapping every handler_local.call() -- prevents exception state
       from leaking to the next request. Drop of tc clears pending exception.
    2. cancel_terminate_execution() called in CpuTimeoutGuard::drop() when
       terminate_execution() was fired -- prevents the V8 terminate flag from
       poisoning subsequent requests on the same isolate.
    3. set_allow_generation_from_strings(false) at Context::new() -- security baseline.
  Proved by:
    - tests/isolate_scope_test.rs SCOPE-01..08 (1000 calls in persistent scope)
    - tests/isolate_endurance_test.rs ENDURE-01..03 (exception recovery, state isolation,
      10+ request endurance)
  Committed: 3f098832 (persistent scope), Phase 40 fixes (TryCatch + cancel)
