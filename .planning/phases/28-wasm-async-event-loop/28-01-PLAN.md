## V8 Async Pattern Analysis

From codebase analysis (src/worker/pool.rs:286, queue.rs:853, v8/module.rs:307, 516, 630):

Current broken pattern (4 locations):
```rust
scope.perform_microtask_checkpoint();  // Only ONE checkpoint
match promise.state() {
    v8::PromiseState::Fulfilled => { /* handle success */ }
    v8::PromiseState::Rejected => { /* handle error */ }
    v8::PromiseState::Pending => {
        return Err(anyhow!("Promise still pending - async execution not fully supported"));
        // ^^ THIS IS THE BUG
    }
}
```

Required working pattern:
```rust
// Loop with timeout until promise resolves
let start = Instant::now();
while promise.state() == v8::PromiseState::Pending {
    if start.elapsed() > timeout {
        return Err(anyhow!("Promise resolution timeout"));
    }
    scope.perform_microtask_checkpoint();
    
    // Pump external async operations via Tokio
    if let Some(handle) = get_worker_runtime() {
        handle.block_on(tokio::task::yield_now());
    }
    
    std::thread::yield_now(); // Cooperate with other threads
}
// Now promise is resolved - extract result
```
