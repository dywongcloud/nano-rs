<objective>
Implement memory monitoring with soft eviction - check memory after each JS call, evict isolates gracefully when limits exceeded, prevent process crashes.

**Purpose:** Production systems must gracefully handle memory pressure without crashing. Soft eviction completes in-flight requests while preventing new ones, LRU eviction targets stateless isolates first (like Cloudflare's approach).

**Output:** Memory monitoring infrastructure, soft eviction policies, LRU cache for isolates, graceful degradation under pressure.
</objective>

## Current Implementation

**Memory Limits (src/worker/limits.rs):**
- MemoryLimiter with per-app heap limits
- OOM detection via V8 near-heap-limit callback
- Hard limit enforcement (immediate termination)

**WorkerPool (src/worker/pool.rs):**
- Thread-local isolates owned by worker threads
- Context reset between requests
- No current memory monitoring after execution

**AppRegistry (src/app/registry.rs):**
- DashMap of hostname → App
- Tracks active apps
- No per-app memory statistics

## Required Approach (Cloudflare-style)

1. **Post-execution memory check:**
   - After every JS call, check V8 heap statistics
   - Compare against soft limit (e.g., 80% of hard limit)
   - Track trend (growing, stable, shrinking)

2. **Soft eviction:**
   - Memory pressure detected → soft eviction mode
   - Allow current requests to complete
   - Block new requests to offending isolate
   - Gradual state drain

3. **LRU eviction:**
   - Track last used timestamp per isolate
   - Evict least recently used stateless isolates
   - Preserve stateful isolates if possible
   - Configurable eviction policy

4. **Memory pressure levels:**
   - Normal: <70% of limit
   - Warning: 70-85% of limit
   - Critical: 85-95% of limit
   - Emergency: >95% of limit (hard eviction)

   ```rust
   fn worker_loop(...) {
       loop {
           match rx.recv().await {
               Some(task) => {
                   // Check if isolate is in draining mode
                   if eviction_manager.is_draining(worker_id) {
                       // Reject new requests
                       let _ = task.response_tx.send(Err(EvictionError::IsolateDraining.into()));
                       continue;
                   }
                   
                   // Execute handler
                   let result = execute_handler(...);
                   
                   // Check memory after execution
                   let snapshot = memory_monitor.check_after(isolate);
                   
                   // Handle pressure
                   match snapshot.pressure_level {
                       MemoryPressureLevel::Normal => {}
                       MemoryPressureLevel::Warning => {
                           // Log warning, may throttle
                           tracing::warn!("Memory warning: {}MB", snapshot.heap_used / 1024 / 1024);
                       }
                       MemoryPressureLevel::Critical | MemoryPressureLevel::Emergency => {
                           // Trigger soft eviction
                           eviction_manager.initiate_soft_eviction(worker_id);
                       }
                   }
                   
                   // Update eviction manager with usage
                   eviction_manager.record_usage(worker_id, IsolateMetadata {
                       last_used: Instant::now(),
                       memory_footprint: snapshot.heap_used,
                       // ...
                   });
                   
                   // Send response
                   let _ = task.response_tx.send(result);
               }
               None => break, // Channel closed
           }
       }
   }
   ```
