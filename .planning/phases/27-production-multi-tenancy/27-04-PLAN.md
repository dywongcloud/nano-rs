## Technical Context

**V8 WebAssembly Support:**
- V8 has built-in WASM engine (no external dependencies)
- rusty_v8 exposes WASM APIs
- WebAssembly.compile() / WebAssembly.instantiate() available
- Memory can be shared between JS and WASM

**Current State:**
- No WASM support currently
- VFS exists for file loading
- Runtime APIs bind WinterCG globals

**Cloudflare Workers Approach:**
- WebAssembly.instantiate() supported
- WASI snapshot preview 1 support
- WASM modules cached and reused
- Same CPU/memory limits apply

**WASM Standards:**
- WebAssembly Core Spec 2.0
- WebAssembly JS API
- WASI Preview 1 (basic filesystem, clock, random)

## Architecture Decisions

1. **Use V8's built-in WASM (not wasmtime):**
   - V8 already has high-performance WASM engine
   - Same isolate = same memory management
   - No additional dependencies
   - rusty_v8 bindings available

2. **WASI support via custom implementation:**
   - wasi-common crate for WASI implementation
   - Map to VFS for filesystem operations
   - Ring for random/getentropy
   - Tokio for clock

3. **WASM in slivers:**
   - Pre-compile WASM modules during snapshot
   - Store compiled code in sliver
   - Faster cold start (no recompilation)

   use crate::vfs::{Vfs, VfsError};
   
   pub struct WasmLoader;
   
   impl WasmLoader {
       /// Load WASM module from filesystem path
       pub fn from_path(path: &str) -> Result<Vec<u8>, WasmError> {
           std::fs::read(path)
               .map_err(|e| WasmError::CompileError(format!("Failed to read {}: {}", path, e)))
       }
       
       /// Load WASM module from VFS
       pub async fn from_vfs(vfs: &dyn Vfs, path: &str) -> Result<Vec<u8>, WasmError> {
           vfs.read_file(path).await
               .map_err(|e| WasmError::CompileError(format!("VFS read failed: {}", e)))
       }
       
       /// Load WASM module from bytes (for inline/embedded)
       pub fn from_bytes(bytes: &[u8]) -> Vec<u8> {
           bytes.to_vec()
       }
       
       /// Validate WASM module without compiling
       pub fn validate(bytes: &[u8]) -> Result<(), WasmError> {
           // Basic magic number and version check
           if bytes.len() < 8 {
               return Err(WasmError::ValidationError("File too small".into()));
           }
           if &bytes[0..4] != b"\0asm" {
               return Err(WasmError::ValidationError("Invalid WASM magic number".into()));
           }
           // Version check (support 1.0 and 2.0)
           let version = u32::from_le_bytes([bytes[4], bytes[5], bytes[6], bytes[7]]);
           if version != 1 && version != 2 {
               return Err(WasmError::ValidationError(format!("Unsupported WASM version: {}", version)));
           }
           Ok(())
       }
   }
   ```

   use v8;
   
   pub struct WasmRuntime<'a> {
       scope: &'a mut v8::HandleScope<'a>,
   }
   
   impl<'a> WasmRuntime<'a> {
       pub fn new(scope: &'a mut v8::HandleScope<'a>) -> Self {
           Self { scope }
       }
       
       /// Compile WASM bytes to a module
       pub fn compile(&mut self, bytes: &[u8]) -> Result<v8::Local<'a, v8::WasmModuleObject>, WasmError> {
           // Validate first
           WasmLoader::validate(bytes)?;
           
           // Create ArrayBuffer from bytes
           let buffer = v8::ArrayBuffer::new_backing_store(self.scope, bytes.len());
           buffer.as_mut_slice().copy_from_slice(bytes);
           let array_buffer = v8::ArrayBuffer::with_backing_store(self.scope, &buffer.make_shared());
           
           // Compile using V8's WebAssembly.compile
           let wasm = v8::Object::new(self.scope);
           let compile_key = v8::String::new(self.scope, "compile").unwrap();
           
           // Get WebAssembly.compile from global
           let global = self.scope.get_current_context().global(self.scope);
           let wasm_key = v8::String::new(self.scope, "WebAssembly").unwrap();
           let wasm_obj = global.get(self.scope, wasm_key.into())
               .ok_or_else(|| WasmError::CompileError("WebAssembly not available".into()))?;
           
           let compile_fn = wasm_obj.to_object(self.scope)
               .and_then(|o| o.get(self.scope, compile_key.into()))
               .ok_or_else(|| WasmError::CompileError("WebAssembly.compile not found".into()))?;
           
           let compile_fn = compile_fn.cast::<v8::Function>();
           let module_promise = compile_fn.call(self.scope, wasm_obj, &[array_buffer.into()])
               .ok_or_else(|| WasmError::CompileError("WebAssembly.compile call failed".into()))?;
           
           // Handle promise resolution synchronously for now
           // In production, this should be async
           self.scope.perform_microtask_checkpoint();
           
           if module_promise.is_promise() {
               let promise = module_promise.cast::<v8::Promise>();
               match promise.state() {
                   v8::PromiseState::Fulfilled => {
                       let module = promise.result(self.scope);
                       Ok(module.cast::<v8::WasmModuleObject>())
                   }
                   v8::PromiseState::Rejected => {
                       let error = promise.result(self.scope);
                       let error_str = error.to_rust_string_lossy(self.scope);
                       Err(WasmError::CompileError(error_str))
                   }
                   v8::PromiseState::Pending => {
                       Err(WasmError::CompileError("WASM compilation still pending".into()))
                   }
               }
           } else {
               Ok(module_promise.cast::<v8::WasmModuleObject>())
           }
       }
       
       /// Instantiate a compiled module
       pub fn instantiate(
           &mut self,
           module: v8::Local<'a, v8::WasmModuleObject>,
           import_object: Option<v8::Local<'a, v8::Object>>
       ) -> Result<v8::Local<'a, v8::WasmInstanceObject>, WasmError> {
           let wasm = v8::Object::new(self.scope);
           let instantiate_key = v8::String::new(self.scope, "instantiate").unwrap();
           
           // Get WebAssembly.instantiate
           let global = self.scope.get_current_context().global(self.scope);
           let wasm_key = v8::String::new(self.scope, "WebAssembly").unwrap();
           let wasm_obj = global.get(self.scope, wasm_key.into())
               .ok_or_else(|| WasmError::InstantiationError("WebAssembly not available".into()))?;
           
           let instantiate_fn = wasm_obj.to_object(self.scope)
               .and_then(|o| o.get(self.scope, instantiate_key.into()))
               .ok_or_else(|| WasmError::InstantiationError("WebAssembly.instantiate not found".into()))?;
           
           let instantiate_fn = instantiate_fn.cast::<v8::Function>();
           let args = if let Some(imports) = import_object {
               vec![module.into(), imports.into()]
           } else {
               vec![module.into()]
           };
           
           let instance_promise = instantiate_fn.call(self.scope, wasm_obj, &args)
               .ok_or_else(|| WasmError::InstantiationError("WebAssembly.instantiate call failed".into()))?;
           
           self.scope.perform_microtask_checkpoint();
           
           if instance_promise.is_promise() {
               let promise = instance_promise.cast::<v8::Promise>();
               match promise.state() {
                   v8::PromiseState::Fulfilled => {
                       let result = promise.result(self.scope);
                       // Result is { module, instance }
                       let instance_key = v8::String::new(self.scope, "instance").unwrap();
                       let instance = result.to_object(self.scope)
                           .and_then(|o| o.get(self.scope, instance_key.into()))
                           .ok_or_else(|| WasmError::InstantiationError("Instance not in result".into()))?;
                       Ok(instance.cast::<v8::WasmInstanceObject>())
                   }
                   v8::PromiseState::Rejected => {
                       let error = promise.result(self.scope);
                       let error_str = error.to_rust_string_lossy(self.scope);
                       Err(WasmError::InstantiationError(error_str))
                   }
                   v8::PromiseState::Pending => {
                       Err(WasmError::InstantiationError("WASM instantiation still pending".into()))
                   }
               }
           } else {
               Ok(instance_promise.cast::<v8::WasmInstanceObject>())
           }
       }
   }
   ```

   //! WebAssembly JavaScript API implementation
   //! 
   //! Implements:
   //! - WebAssembly.compile(bytes) -> Promise<Module>
   //! - WebAssembly.instantiate(moduleOrBytes, imports) -> Promise<Instance>
   //! - WebAssembly.Module class
   //! - WebAssembly.Instance class
   //! - WebAssembly.Memory class
   //! - WebAssembly.Table class
   //! - WebAssembly.Global class
   //! - WebAssembly.CompileError, RuntimeError
   
   use v8;
   
   pub struct WebAssemblyAPI;
   
   impl WebAssemblyAPI {
       /// Bind WebAssembly global to context
       pub fn bind(scope: &mut v8::HandleScope, context: v8::Local<v8::Context>) {
           let wasm_obj = v8::Object::new(scope);
           
           // WebAssembly.compile
           let compile_fn = v8::FunctionTemplate::new(scope, Self::compile_callback);
           let compile_name = v8::String::new(scope, "compile").unwrap();
           wasm_obj.set(scope, compile_name.into(), compile_fn.get_function(scope).unwrap().into());
           
           // WebAssembly.instantiate
           let instantiate_fn = v8::FunctionTemplate::new(scope, Self::instantiate_callback);
           let instantiate_name = v8::String::new(scope, "instantiate").unwrap();
           wasm_obj.set(scope, instantiate_name.into(), instantiate_fn.get_function(scope).unwrap().into());
           
           // WebAssembly.validate
           let validate_fn = v8::FunctionTemplate::new(scope, Self::validate_callback);
           let validate_name = v8::String::new(scope, "validate").unwrap();
           wasm_obj.set(scope, validate_name.into(), validate_fn.get_function(scope).unwrap().into());
           
           // WebAssembly.Module
           let module_ctor = Self::create_module_constructor(scope);
           let module_name = v8::String::new(scope, "Module").unwrap();
           wasm_obj.set(scope, module_name.into(), module_ctor.into());
           
           // WebAssembly.Instance
           let instance_ctor = Self::create_instance_constructor(scope);
           let instance_name = v8::String::new(scope, "Instance").unwrap();
           wasm_obj.set(scope, instance_name.into(), instance_ctor.into());
           
           // WebAssembly.Memory
           let memory_ctor = Self::create_memory_constructor(scope);
           let memory_name = v8::String::new(scope, "Memory").unwrap();
           wasm_obj.set(scope, memory_name.into(), memory_ctor.into());
           
           // WebAssembly.CompileError
           let compile_error = Self::create_error_constructor(scope, "CompileError");
           let compile_error_name = v8::String::new(scope, "CompileError").unwrap();
           wasm_obj.set(scope, compile_error_name.into(), compile_error.into());
           
           // WebAssembly.RuntimeError
           let runtime_error = Self::create_error_constructor(scope, "RuntimeError");
           let runtime_error_name = v8::String::new(scope, "RuntimeError").unwrap();
           wasm_obj.set(scope, runtime_error_name.into(), runtime_error.into());
           
           // Set WebAssembly global
           let global = context.global(scope);
           let wasm_name = v8::String::new(scope, "WebAssembly").unwrap();
           global.set(scope, wasm_name.into(), wasm_obj.into());
       }
       
       fn compile_callback(
           scope: &mut v8::HandleScope,
           args: v8::FunctionCallbackArguments,
           mut retval: v8::ReturnValue,
       ) {
           // Extract bytes argument
           let buffer = args.get(0);
           if !buffer.is_array_buffer() && !buffer.is_typed_array() {
               let msg = v8::String::new(scope, "WebAssembly.compile: argument must be BufferSource").unwrap();
               let error = v8::Exception::type_error(scope, msg);
               scope.throw_exception(error);
               return;
           }
           
           // Get bytes
           let bytes = if buffer.is_array_buffer() {
               let ab = buffer.cast::<v8::ArrayBuffer>();
               let store = ab.get_backing_store();
               store.as_slice().to_vec()
           } else {
               let ta = buffer.cast::<v8::TypedArray>();
               let ab = ta.buffer(scope).unwrap();
               let store = ab.get_backing_store();
               store.as_slice().to_vec()
           };
           
           // Validate
           if let Err(e) = WasmLoader::validate(&bytes) {
               let msg = v8::String::new(scope, &format!("WebAssembly.compile: {}", e)).unwrap();
               let error = v8::Exception::error(scope, msg);
               scope.throw_exception(error);
               return;
           }
           
           // Use native V8 WebAssembly.compile (already available)
           // The native implementation will handle the actual compilation
           // We just validated and prepared the bytes
           let global = scope.get_current_context().global(scope);
           let wasm_key = v8::String::new(scope, "WebAssembly").unwrap();
           let wasm_obj = global.get(scope, wasm_key.into()).unwrap();
           
           let compile_key = v8::String::new(scope, "compile").unwrap();
           let compile_fn = wasm_obj.to_object(scope).unwrap()
               .get(scope, compile_key.into()).unwrap()
               .cast::<v8::Function>();
           
           // Create ArrayBuffer from bytes
           let store = v8::ArrayBuffer::new_backing_store(scope, bytes.len());
           store.as_mut_slice().copy_from_slice(&bytes);
           let ab = v8::ArrayBuffer::with_backing_store(scope, &store.make_shared());
           
           let result = compile_fn.call(scope, wasm_obj, &[ab.into()]);
           if let Some(r) = result {
               retval.set(r);
           }
       }
       
       fn instantiate_callback(
           scope: &mut v8::HandleScope,
           args: v8::FunctionCallbackArguments,
           mut retval: v8::ReturnValue,
       ) {
           // Similar pattern to compile
           // Handle both (bytes, imports) and (module, imports) signatures
           // Delegate to native V8 implementation
       }
       
       fn validate_callback(
           scope: &mut v8::HandleScope,
           args: v8::FunctionCallbackArguments,
           mut retval: v8::ReturnValue,
       ) {
           let buffer = args.get(0);
           let bytes = if buffer.is_array_buffer() {
               let ab = buffer.cast::<v8::ArrayBuffer>();
               let store = ab.get_backing_store();
               store.as_slice().to_vec()
           } else {
               let msg = v8::String::new(scope, "WebAssembly.validate: argument must be BufferSource").unwrap();
               let error = v8::Exception::type_error(scope, msg);
               scope.throw_exception(error);
               return;
           };
           
           let valid = WasmLoader::validate(&bytes).is_ok();
           retval.set(v8::Boolean::new(scope, valid).into());
       }
       
       fn create_module_constructor(scope: &mut v8::HandleScope) -> v8::Local<v8::Function> {
           let tpl = v8::FunctionTemplate::new(scope, |scope: &mut v8::HandleScope, 
               args: v8::FunctionCallbackArguments, 
               mut retval: v8::ReturnValue| {
               // Module constructor - delegates to native
           });
           tpl.get_function(scope).unwrap()
       }
       
       fn create_instance_constructor(scope: &mut v8::HandleScope) -> v8::Local<v8::Function> {
           let tpl = v8::FunctionTemplate::new(scope, |scope: &mut v8::HandleScope,
               args: v8::FunctionCallbackArguments,
               mut retval: v8::ReturnValue| {
               // Instance constructor - delegates to native
           });
           tpl.get_function(scope).unwrap()
       }
       
       fn create_memory_constructor(scope: &mut v8::HandleScope) -> v8::Local<v8::Function> {
           // WebAssembly.Memory constructor
           let tpl = v8::FunctionTemplate::new(scope, |_scope: &mut v8::HandleScope,
               _args: v8::FunctionCallbackArguments,
               _retval: v8::ReturnValue| {
               // Delegates to native V8 implementation
           });
           tpl.get_function(scope).unwrap()
       }
       
       fn create_error_constructor(scope: &mut v8::HandleScope, name: &str) -> v8::Local<v8::Function> {
           let tpl = v8::FunctionTemplate::new(scope, |_scope: &mut v8::HandleScope,
               _args: v8::FunctionCallbackArguments,
               _retval: v8::ReturnValue| {
           });
           let name_str = v8::String::new(scope, name).unwrap();
           tpl.set_class_name(name_str);
           tpl.get_function(scope).unwrap()
       }

   //! WASM module support in sliver snapshots
   //!
   //! Enables pre-compiled WASM modules to be stored in slivers
   //! for faster cold starts (skip compilation step).
   
   use std::collections::HashMap;
   use dashmap::DashMap;
   
   /// Cache of compiled WASM modules
   pub struct SliverWasmCache {
       /// Path -> compiled module bytes
       modules: DashMap<String, CompiledWasmModule>,
   }
   
   pub struct CompiledWasmModule {
       /// Original WASM bytes hash (for verification)
       pub source_hash: [u8; 32],
       /// V8 compiled module serialized bytes
       pub compiled_bytes: Vec<u8>,
       /// Module size in bytes
       pub size: usize,
       /// Compilation timestamp
       pub compiled_at: std::time::SystemTime,
   }
   
   impl SliverWasmCache {
       pub fn new() -> Self {
           Self {
               modules: DashMap::new(),
           }
       }
       
       /// Check if module is cached
       pub fn get(&self, path: &str) -> Option<CompiledWasmModule> {
           self.modules.get(path).map(|m| m.clone())
       }
       
       /// Store compiled module
       pub fn insert(&self, path: String, module: CompiledWasmModule) {
           self.modules.insert(path, module);
       }
       
       /// Serialize all cached modules for sliver storage
       pub fn serialize(&self) -> Vec<u8> {
           // Use bincode or similar for efficient serialization
           let modules: HashMap<String, CompiledWasmModule> = self.modules
               .iter()
               .map(|e| (e.key().clone(), e.value().clone()))
               .collect();
           
           bincode::serialize(&modules)
               .expect("Failed to serialize WASM cache")
       }
       
       /// Deserialize from sliver
       pub fn deserialize(bytes: &[u8]) -> Result<Self, WasmError> {
           let modules: HashMap<String, CompiledWasmModule> = bincode::deserialize(bytes)
               .map_err(|e| WasmError::CompileError(format!("Failed to deserialize WASM cache: {}", e)))?;
           
           let cache = Self::new();
           for (path, module) in modules {
               cache.modules.insert(path, module);
           }
           
           Ok(cache)
       }
       
       /// Get all module paths
       pub fn module_paths(&self) -> Vec<String> {
           self.modules.iter().map(|e| e.key().clone()).collect()
       }
       
       /// Total size of all cached modules
       pub fn total_size(&self) -> usize {
           self.modules.iter().map(|e| e.value().size).sum()
       }
       
       /// Module count
       pub fn module_count(&self) -> usize {
           self.modules.len()
       }
   }
   
   impl Clone for CompiledWasmModule {
       fn clone(&self) -> Self {
           Self {
               source_hash: self.source_hash,
               compiled_bytes: self.compiled_bytes.clone(),
               size: self.size,
               compiled_at: self.compiled_at,
           }
       }
   }
   
   /// Helper to compile and cache WASM modules during sliver creation
   pub fn compile_and_cache(
       scope: &mut v8::HandleScope,
       vfs: &dyn crate::vfs::Vfs,
       path: &str,
       cache: &SliverWasmCache,
   ) -> Result<(), WasmError> {
       // Load WASM bytes from VFS
       let bytes = futures::executor::block_on(vfs.read_file(path))
           .map_err(|e| WasmError::CompileError(format!("Failed to read {}: {}", path, e)))?;
       
       // Compile using WasmRuntime
       let mut runtime = WasmRuntime::new(scope);
       let module = runtime.compile(&bytes)?;
       
       // Serialize compiled module (V8 API)
       // Note: V8's module serialization requires specific API
       // For now, store source hash and bytes
       // Future: Use v8::WasmModuleObject::serialize()
       
       use sha2::{Sha256, Digest};
       let mut hasher = Sha256::new();
       hasher.update(&bytes);
       let source_hash = hasher.finalize().into();
       
       let compiled = CompiledWasmModule {
           source_hash,
           compiled_bytes: bytes, // Store source for now (compilation on restore)
           size: bytes.len(),
           compiled_at: std::time::SystemTime::now(),
       };
       
       cache.insert(path.to_string(), compiled);
       
       Ok(())
   }
   
   /// Restore WASM module from cache
   pub fn restore_from_cache(
       scope: &mut v8::HandleScope,
       path: &str,
       cache: &SliverWasmCache,
   ) -> Result<v8::Local<v8::WasmModuleObject>, WasmError> {
       match cache.get(path) {
           Some(compiled) => {
               // Verify source hash if needed
               // For now, just recompile from stored bytes
               // Future: Use serialized compiled module
               let mut runtime = WasmRuntime::new(scope);
               runtime.compile(&compiled.compiled_bytes)
           }
           None => Err(WasmError::CompileError(format!("WASM module not in cache: {}", path))),
       }
   }
