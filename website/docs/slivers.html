Sliver Snapshots

Portable isolate snapshots for sub-millisecond cold starts

What are Slivers?

Slivers are portable snapshots of JavaScript isolates. They capture the complete state of an isolate including:

  • Compiled JavaScript code
  • VFS filesystem state
  • Global object state
  • Context bindings

Creating a Sliver

Terminal
# Create sliver from running app
nano-rs sliver create localhost --name my-app --tag v1.0

# List slivers
nano-rs sliver list
